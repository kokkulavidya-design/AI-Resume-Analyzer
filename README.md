📄 AI Resume Analyzer

![Status](https://img.shields.io/badge/Project-Active-brightgreen)

An interactive beginner-friendly AI web application built using Python and Streamlit that analyzes uploaded resumes (PDF format) and 
provides instant insights such as skills detection, ATS scoring, improvement suggestions, and job role recommendations.

 🚀 Features

📂 Upload resume in PDF format  
📄 Extract text from resume automatically using PyPDF2  
💡 Detect key technical and soft skills  
🎯 Generate ATS Score based on skills  
📊 Visualize detected skills using bar chart (Matplotlib)  
🚀 Provide resume improvement suggestions  
💼 Recommend suitable job roles based on skills found  


 🛠️ Tech Stack

Python 🐍  
Streamlit 🎈  
PyPDF2 📄  
Matplotlib 📊  

 📌 How It Works

1. Upload a resume in PDF format  
2. Text is extracted from the resume  
3. Skills are matched using a predefined list  
4. ATS score is calculated based on detected skills  
5. Suggestions are generated if important sections are missing  
6. Skill chart is displayed  
7. Job role is recommended based on skills  

 📊 Skills Detected

- Python  
- Java  
- C  
- C++  
- HTML  
- CSS  
- JavaScript  
- SQL  
- Machine Learning  
- Communication  

 🎯 ATS Scoring Logic

- Each detected skill adds points to the score  
- Maximum score = 100%  
- Higher score = better resume strength  

 💼 Job Role Prediction

- Python + SQL → Python Developer  
- HTML + CSS → Frontend Developer  
- Machine Learning → ML Engineer  
- Java → Java Developer  
- Otherwise → Skill data insufficient


 🚀 How to Run

```bash
git clone https://github.com/Kokkulavidya-design/Ai-resume-analyzer.git

cd Ai-resume-analyzer

pip install streamlit PyPDF2 matplotlib

streamlit run app.py
```
